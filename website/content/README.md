```bash
cd website

```

Open http://localhost:8080

---

## Update content → website auto-syncs

1. Edit any `.md` file in this repo root
2. Push to `main`
3. GitHub Action runs `prepare-deploy.js` and syncs `website/content/`
4. GitHub Pages serves the updated site (1–2 min)

**Manual sync (optional):**
```bash
cd website
node prepare-deploy.js stack-ready
```

---

## GitHub Pages setup (one-time)

1. Wait for **Deploy StackReady to GitHub Pages** to succeed (Actions tab)
2. Settings → **Pages** → Source: **Deploy from a branch**
3. Branch: **`gh-pages`** → Folder: **`/ (root)`** → Save

Live URL: **https://jerryd24.github.io/stack-ready/**
